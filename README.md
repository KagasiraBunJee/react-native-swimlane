# react-native-swimlane

Prototype for kanban style table style with sections

## Installation

```sh
npm install react-native-swimlane
```

## Usage

```js
import Swimlane from "react-native-swimlane";

// ...

const result = await Swimlane.multiply(3, 7);
```
